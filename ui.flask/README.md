# linux-kernel-cves ui.flask
> A flask UI to navigate data from linux-kernel-cves

## Setting up the flask ui
``` bash
# clone the linux-kernel-cves repo
git clone https://github.com/nluedtke/linux_kernel_cves.git

# install flask
pip install flask

# move to ui folder
cd ui.flask

# run
python a.py
```
